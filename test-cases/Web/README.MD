# 🌐 Web Testing

## 📌 Deskripsi

Folder ini berisi kumpulan **test cases untuk pengujian aplikasi berbasis web** yang disusun berdasarkan module atau fitur tertentu.

Seluruh dokumentasi pengujian ditulis dalam format **Markdown** untuk memudahkan pembacaan langsung di GitHub tanpa perlu membuka file eksternal.

---

## 📂 Struktur Folder

```bash id="3dh36q"
web/
├── login-form/
│   ├── test-suite-login.md
│   ├── test-scenario.md
│   ├── test-case.md
│   ├── bug-report.md
│   └── evidence/
```

---

## 🧪 Artefak Pengujian

Setiap module dalam folder ini terdiri dari:

* **Test Suite**
  Menjelaskan strategi dan ruang lingkup pengujian

* **Test Scenario**
  Daftar skenario pengujian berdasarkan berbagai kondisi

* **Test Case**
  Langkah detail pengujian dalam format tabel Markdown

* **Bug Report**
  Dokumentasi bug yang ditemukan selama testing

* **Evidence**
  Bukti berupa screenshot hasil pengujian

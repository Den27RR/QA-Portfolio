# 🧪 Test Cases

## 📌 Deskripsi

Folder ini berisi kumpulan **test cases** yang digunakan untuk menguji berbagai fitur aplikasi berdasarkan platform yang berbeda, yaitu **Web** dan **Mobile**.

Seluruh test case disusun menggunakan format **Markdown** agar mudah dibaca langsung di GitHub dan mengikuti praktik dokumentasi yang umum digunakan dalam proses **Quality Assurance (QA)**.

---

## 📂 Struktur Folder

```bash
test-cases/
├── Web/
│   └── (web test modules)
│
└── Mobile/
    └── (mobile test modules - coming soon)
```

---

## 🌐 Web Testing

Berisi pengujian untuk aplikasi berbasis web, yang mencakup berbagai module seperti:

* Login Form
* Register
* Cart
* Checkout

Setiap module terdiri dari:

* Test Suite
* Test Scenario
* Test Case
* Bug Report
* Evidence

---

## 📱 Mobile Testing

Folder ini disiapkan untuk pengujian aplikasi berbasis mobile (Android/iOS).

*(Akan dikembangkan pada tahap selanjutnya)*

---

## 🎯 Tujuan

* Mengorganisir test case berdasarkan platform
* Mempermudah navigasi dan eksplorasi repository
* Mendukung pengembangan repository secara scalable
* Menyerupai struktur project QA di dunia industri

---

## 🧠 Pendekatan

Struktur ini dibuat untuk memisahkan pengujian berdasarkan platform (**Web** dan **Mobile**) sehingga:

* Lebih mudah dalam pengelolaan
* Lebih jelas dalam dokumentasi
* Lebih fleksibel untuk pengembangan ke depan

---

> Folder ini akan terus diperbarui dengan penambahan module dan peningkatan kualitas test cases.

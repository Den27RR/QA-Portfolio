## 📌 Deskripsi Proyek

Repository ini berisi portofolio **manual testing** yang berfokus pada pengujian fitur **Login** pada aplikasi web SauceDemo.

Proyek ini dibuat untuk menunjukkan kemampuan dalam:

* Test Planning
* Test Scenario
* Test Case Design
* Bug Reporting

Seluruh proses pengujian disusun menyerupai praktik kerja **Quality Assurance (QA)** di dunia industri.

---

## 🎯 Tujuan Pengujian

* Memastikan fitur login berjalan sesuai fungsi yang diharapkan
* Mengidentifikasi bug atau error pada proses autentikasi
* Memvalidasi sistem terhadap input valid dan tidak valid

---

## 🔍 Ruang Lingkup (Scope)

* Validasi input username
* Validasi input password
* Verifikasi error message
* Perilaku sistem saat proses login

---

## 🧪 Jenis Pengujian (Testing Types)

* Functional Testing
* Negative Testing
* Boundary Testing
* Basic Security Testing
